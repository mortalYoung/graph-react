# react-graph

## Getting Started

本项目基于 `dumi`, `mxgraph`, `react` 实现

---
Install dependencies,

```bash
$ yarn i
```

Start the dev server,

```bash
$ yarn start
```

Build documentation,

```bash
$ yarn docs:build
```

Build library via `father-build`,

```bash
$ yarn build
```

Build github pages,

```bash
$ yarn deploy
```
